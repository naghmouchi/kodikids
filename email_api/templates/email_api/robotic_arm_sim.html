Documentation Bras

Commandes Python (Simulées) :

  • arm.close(doigt)
    Ferme un doigt (0=Pouce, 1=Index, etc.).
  • arm.open(doigt)
    Ouvre un doigt.
  • arm.grip()
    Ferme tous les doigts (saisir).
  • arm.release()
    Ouvre tous les doigts (lâcher).
  • arm.sleep(secondes)
    Pause.

Index des doigts :
0: Pouce, 1: Index, 2: Majeur, 3: Annulaire, 4: Auriculaire

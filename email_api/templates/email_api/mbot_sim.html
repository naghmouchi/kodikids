Documentation mBot

Commandes Python (Simulées) :

  • mbot.forward(1)
    Avance pendant 1 seconde.
  • mbot.backward(1)
    Recule pendant 1 seconde.
  • mbot.turn_left(0.5)
    Tourne à gauche (0.5s).
  • mbot.turn_right(0.5)
    Tourne à droite (0.5s).
  • mbot.led(255, 0, 0)
    Allume les LEDs (Rouge, Vert, Bleu).
  • mbot.led(0, 0, 0)
    Éteint les LEDs.

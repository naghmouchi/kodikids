Documentation IoT

Bibliothèque `iot` (Simulée) :

  • iot.connect("SSID", "PWD")
    Connecte l'ESP32 au WiFi.
  • iot.read_temp()
    Lit la température (20-35°C).
  • iot.send("topic", valeur)
    Envoie une donnée au Cloud (MQTT).
  • iot.fan_on() / off()
    Active/Désactive le ventilateur.
  • iot.sleep(secondes)
    Pause le programme.
